## About

Collect meme coins from the conveyor belt, place them on pedestals to earn passive income, merge matching coins to level up, and sell for profit. Features include:

- 21 unique coin types across 7 tiers
- 6 rarity levels (Common to Mythic)
- 4 mutations (Gold, Diamond, Dark Matter, Rainbow)
- 3 floors with 36 total slots
- Rebirth system for permanent multipliers
- Premium shop with ETH payments on Base

## Tech Stack

- Frontend: Vanilla JS + Canvas
- Blockchain: Base (Ethereum L2)
- Smart Contract: Solidity 0.8.19
