# Steal A Coin

A pixel-art idle tycoon game for Yandex Games.

## Play Now

**[Play on Yandex Games](https://yandex.ru/games/app/XXXXXX)** *(link will be updated after publication)*

## About

Collect coins from the conveyor belt, place them on pedestals to earn passive income, merge matching coins to level up, and sell for profit!

### Features

- 21 unique coin types across 7 tiers
- 6 rarity levels (Common to Mythic)
- 4 mutations (Gold, Diamond, Dark Matter, Rainbow)
- 3 floors with 36 total slots
- Rebirth system for permanent multipliers
- Cloud saves via Yandex Games
- Rewarded ads for bonuses

### Controls

- **WASD** - Move
- **E** - Interact (pick up / place)
- **TAB** - Shop
- **Q** - Stats
- **R** - Rebirth menu
- **C** - Collection
- **G** - Watch ad for bonus
- **M** - Toggle sound
- **1/2/3** - Switch floors

## Tech Stack

- Vanilla JavaScript + Canvas
- Yandex Games SDK

## Local Development

```bash
node server.js
```
